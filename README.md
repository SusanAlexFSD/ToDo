To‑Do List App

A simple and functional to‑do list application built with HTML, CSS, and JavaScript. The app allows users to add, complete, and remove tasks, focusing on clear core functionality and straightforward task management.
Features

    Add new tasks

    Mark tasks as completed

    Remove tasks from the list

    Lightweight and fast — no frameworks required

Tech Stack

    HTML

    CSS

    JavaScript

Status

The main functionality is in place, but the app is still a work in progress. Planned improvements include:

    Enhancing the UI for a cleaner, more modern look

    Adding more features such as task categories, due dates, or priority levels

    Improving overall user experience and layout

    Introducing animations or transitions for a smoother feel

How It Works

    Enter a task into the input field.

    Add it to the list with a single click.

    Mark tasks as completed or remove them entirely.
